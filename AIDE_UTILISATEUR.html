🚀 1. Démarrage

Installer l'application sur la tablette

Ouvrir l'URL dans le navigateur

Sur Android : ouvrir Chrome. Sur iPad : ouvrir Safari.
Saisir l'adresse fournie par le professeur.

Installer l'application

Android (Chrome) : Menu ⋮ → « Ajouter à l'écran d'accueil »
iPad (Safari) : Bouton partage ↑ → « Sur l'écran d'accueil »

Lancer VB EPS depuis l'écran d'accueil

L'application s'ouvre en plein écran, sans barre du navigateur.
Elle fonctionne sans connexion internet une fois installée.

ℹ️
La première ouverture nécessite une connexion pour télécharger l'application. Ensuite, tout fonctionne en gymnase ou sur terrain sans WiFi.

🗺️ 2. Navigation dans l'application

L'application se compose de 5 écrans parcourus dans l'ordre :

🏠 Accueil → 👤 Identités → ⚙️ Config mode → 📊 Observation → 📤 Export

Barre du haut (topbar)

BoutonAction
📋Accéder aux observations en cours (liste des séances archivées)
⚙️Ouvrir les réglages (thème, chrono, vibration, mode prof)
← RetourRevenir à l'écran précédent sans perdre les données
⚠️
Le bouton « Nouvelle séance (vider cache) » efface toutes les données de la séance en cours. À utiliser seulement en début de nouvelle séance.

À l'écran Config mode, choisir :

📊 4. Faire une observation

Déroulement d'une séance complète

Saisir la classe (écran Accueil)

La date et l'heure se remplissent automatiquement. Vous pouvez les désactiver pour les saisir manuellement.

Saisir les identités (écran Identités)

Prénom et Nom de l'observateur (l'élève qui observe) et du joueur observé.

Configurer le mode (écran Config)

Choisir le niveau (Collège/Lycée), le type (Individuel/Collectif) et le(s) rôle(s) à observer.

Observer et noter (écran Observation)
  • Le chronomètre démarre automatiquement (réglable : 3 à 10 min).
  • Appuyer sur le bouton correspondant au geste réalisé.
  • Chaque appui enregistre 1 action et la fait vibrer pour confirmer.
  • Le compteur et le total en temps réel s'affichent à l'écran.
Archiver ou terminer

✅ Nouvelle observation — archive ce joueur et retourne saisir un autre joueur (jusqu'à 5 par séance).

📤 Enregistrer la séance — termine et passe à l'export.

💡 Astuce chrono : Appuyer sur le chronomètre pour le mettre en pause / reprendre. Il s'arrête automatiquement à zéro mais l'observation continue.

Limite de la séance

L'application gère jusqu'à 5 observations par séance pour garantir la lisibilité du QR code à l'export. Au-delà, enregistrer la séance en cours, exporter, puis démarrer une nouvelle séance.

Dictée vocale (optionnelle)

Activer dans ⚙️ Réglages → Dictée vocale. Dire à voix haute le geste observé :

DireAction enregistrée
Serveur
« tennis »4 pts — Service tennis réussi
« cuillère »3 pts — Cuillère réussie
« passe »1 pt — Passe haute
« raté » / « zéro »0 pt — Service raté
Renvoyeur
« smashé »4 pts — Smash réussi
« placé »3 pts — Renvoi placé
« dos »1 pt — Dos au filet
🎤
La dictée vocale nécessite l'autorisation du microphone. Elle fonctionne hors ligne une fois accordée.

📤 6. Export CSV & QR code

Export CSV (tableur)

Appuyer sur « Télécharger CSV »

Le fichier se nomme automatiquement : EPS_Volleyball_4eA_2025-04-10.csv

Ouvrir dans Excel ou LibreOffice

Le séparateur est le point-virgule (;), compatible avec Excel en français.
Si les colonnes ne se séparent pas, aller dans Données → Convertir et choisir « point-virgule ».

Colonnes du fichier CSV

ColonneContenu
joueurPrénom Nom du joueur observé
classeClasse saisie à l'accueil
observateurPrénom Nom de l'élève observateur
date / heureDate et heure de la séance
mode / niveauEx. : « Individuel » / « Collège »
roleEx. : « Serveur », « Renvoyeur »…
total_pts_*Total de points par rôle
pts_moy_*Points moyens par action
detail_scoresListe brute de tous les scores (désactivée par défaut)
ℹ️
Les colonnes exportées sont configurables en mode Professeur (⚙️ → ADMINPROF → section Export).

QR code (partage rapide)

Un QR code est généré automatiquement à partir des données de la séance.

  • Scanner avec un autre smartphone ou tablette pour recevoir les données.
  • Maximum 5 observations par QR code (limite de lisibilité).
  • Fonctionne 100 % hors ligne — aucune connexion nécessaire.

💡 8. Astuces & questions fréquentes

L'application ne s'installe pas sur iPad

L'installation PWA sur iOS nécessite Safari (pas Chrome ni Firefox). Ouvrir l'URL dans Safari, puis bouton de partage ↑ → « Sur l'écran d'accueil ».

Les données ont disparu après fermeture

Les données sont sauvegardées dans la mémoire locale de la tablette (localStorage). Elles persistent entre les sessions sauf si :

  • Le bouton « Nouvelle séance (vider cache) » a été appuyé.
  • Le navigateur a été configuré pour effacer les données à la fermeture.
  • La mémoire de l'appareil a été nettoyée manuellement.

Le CSV ne s'ouvre pas correctement dans Excel

Excel attend parfois un séparateur virgule. Solution : dans Excel → Données → Données externes → À partir du texte, choisir séparateur : point-virgule. Sur Mac : importer via « Données → Nouveau texte CSV ».

Organiser une évaluation en classe entière

✅ Workflow recommandé :
1 tablette = 1 observateur. Chaque observateur observe 1 joueur pendant la séance (5 min).
À la fin, chaque tablette exporte son CSV. Le professeur collecte les fichiers (clé USB ou QR code).

Activer / désactiver le mode sombre

⚙️ Réglages → Mode sombre. Le thème est mémorisé entre les sessions.

Changer la couleur de l'interface

⚙️ Réglages → Couleur du thème : Violet (défaut), Vert, Orange, Bleu.

La dictée vocale ne fonctionne pas

Vérifier que :

  • L'autorisation microphone est accordée dans les réglages du navigateur.
  • La langue de l'appareil est réglée sur le français.
  • Le toggle « Dictée vocale » est bien activé dans ⚙️ Réglages.
ℹ️
La dictée vocale utilise l'API du système et peut être limitée sur certaines versions d'iOS.
